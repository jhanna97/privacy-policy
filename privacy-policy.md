# Privacy Policy

## Overview
This is a personal event planning application that integrates with Pinterest to display board inspiration for hens parties and events.

## Data Collection
- We do not collect, store, or share any personal user data
- The application only retrieves publicly available Pinterest board data that you authorize access to
- No cookies, tracking, or analytics are implemented

## Data Usage
- Pinterest board data is displayed for personal event planning purposes only
- No data is shared with third parties
- All data remains under your control

## Changes to This Policy
We may update this privacy policy occasionally. Changes will be posted to this page.
